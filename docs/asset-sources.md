## ðŸŽ¯ Asset Specifications Summary

### GIF Optimization
- **Format**: GIF89a (supports transparency)
- **Color Depth**: 256 colors max (8-bit)
- **Size Goals**:
  - Small icons/badges: < 50KB
  - Buttons: < 100KB
  - Background tiles: < 200KB
  - Large animations: < 500KB
- **Optimization**: Use tools like ezgif.com if needed

### Image Formats
- **Icons**: PNG with transparency
- **BSOD**: PNG or JPG (no transparency needed)
- **Badges**: GIF (for animations) or PNG (static)

### Audio Formats
- **MIDI**: .mid or .midi
- **WAV**: Uncompressed (if used)
- **MP3**: Compressed alternative (if MIDI blocked)
